# PartyPilot

AI-powered event planning and booking system for social outings.

## Quick Start

### Prerequisites
- Node.js 18+
- pnpm (or npm/yarn)

### Installation

```bash
# Install dependencies
pnpm install

# Run development server (frontend)
pnpm dev

# Run API server (in separate terminal)
cd apps/api
pnpm dev
```

## Project Structure

- `apps/web` - Next.js frontend
- `apps/api` - Express API backend
- `docs/` - Documentation

## Development

The frontend runs on `http://localhost:3000` and the API on `http://localhost:3001`.
